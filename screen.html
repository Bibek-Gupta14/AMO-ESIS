Screen Reader Access

Assistive Technologies are meant to cater to the needs of people with various impairments like Visual, Hearing, Mobility, and Learning impairment. An assistive technology helps differently abled people in rendering or viewing a webpage. Screen Readers are assistive devices that convert text into synthesized speech to facilitate visually impaired users to listen and comprehend the content on a webpage. Following links can be visited to get more information about assistive technologies and screen readers:

  • http://www.screenreader.net
  • http://www.freedomscientific.com
  • http://www.gwmicro.com
  • http://www.spectronicsinoz.com

Accessibility Controls

This portal is equipped with built-in accessibility controls located in the top bar to adjust the reading layout according to your visual preference:

  • Increase Text size (↑): Increases the page text size step-by-step up to 130% of the default size.
  • Decrease Text size (↓): Decreases the page text size step-by-step down to 85% of the default size.
  • Text Spacing: Adjusts letter and word spacing to make reading comfortable.
  • Line Height: Increases the height between lines of text for better paragraph readability.
  • Theme Contrast Controls: Switches between normal mode, dark mode (dark background), and white contrast (high contrast light background).
  • Reset Text: Instantly resets all custom font sizes, text spacing, and line heights back to the default settings.

Screen Reader Access

The "Guidelines for Indian Government Websites" website complies with World Wide Web Consortium (W3C) Web Content Accessibility Guidelines (WCAG) 2.0 level AA. This will enable people with visual impairments access the website using assistive technologies, such as screen readers. The information of the website is accessible with different screen readers, such as JAWS, NVDA, SAFA, Supernova and Window‑Eyes.
